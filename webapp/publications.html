Research Publications

Below is a list of my published research papers in the fields of Cryptography, Machine Learning, and Data Science.

  • A Lightweight Cryptographic Solution for 6LoWPAN Protocol Stack

    Published in: Advances in Intelligent Systems and Computing, Springer (2018)

    Conference: Science and Information (Computing Conference) 2018, London, UK

    This paper presents an implementation of a lightweight 6LoWPAN protocol stack using efficient ciphers designed for IoT devices, with analysis against major cryptographic attacks.

  • Combating Bias in Recidivism Algorithms – An AI Approach

    Published in: International Journal of Advanced Computer Science and Applications (IJACSA), Vol. 16, Issue 6, 2025

    This paper explores algorithmic fairness and bias mitigation in AI-based criminal recidivism prediction models using explainable machine learning techniques.

  • AI-Based Cybersecurity Threat Detection using Ensemble Learning Models

    Published in: International Journal of Advanced Computer Science and Applications (IJACSA), Vol. 16, Issue 7, 2025

    This work proposes an ensemble learning framework for identifying emerging cyber threats, combining Random Forest and Deep Neural Network architectures.

  • Analysis and Evaluation of Modern Lightweight Cryptographic Algorithms – Standards, Hardware Implementation and Security Considerations

    Published in: International Journal of Computer Applications (IJCA), Vol. 187, Issue 37, 2025

    This paper analyzes lightweight cryptographic standards such as PRESENT, SPECK, and SIMON, evaluating their security and hardware performance trade-offs for IoT environments.
