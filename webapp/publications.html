Research Publications

Below is a list of my published research papers in the fields of Cryptography, Machine Learning, and Data Science.

  • A Lightweight Cryptographic Solution for 6LoWPAN Protocol Stack

    Published in: Advances in Intelligent Systems and Computing, Springer (2018)

    Conference: Computing Conference 2018, London, UK

    This paper presents an implementation of a lightweight 6LoWPAN protocol stack using efficient ciphers designed for IoT devices, with analysis against major cryptographic attacks.

  • Combating Bias in Recidivism Algorithms – An AI Approach

    Published in: IJACSA, Vol. 16, Issue 6, 2025

    Explores algorithmic fairness and bias mitigation in AI-based criminal recidivism prediction models using explainable ML techniques.

  • AI-Based Cybersecurity Threat Detection using Ensemble Learning Models

    Published in: IJACSA, Vol. 16, Issue 7, 2025

    Proposes an ensemble learning framework for identifying emerging cyber threats, combining Random Forest and Deep Neural Network architectures.

  • Analysis and Evaluation of Modern Lightweight Cryptographic Algorithms – Standards, Hardware Implementation and Security Considerations

    Published in: IJCA, Vol. 187, Issue 37, 2025

    Analyzes lightweight cryptographic standards such as PRESENT, SPECK, and SIMON, evaluating their security and hardware performance trade-offs for IoT environments.

  • Blockchain and AI for Secure IoT Data Exchange: A Lightweight Hybrid Model

    Published in: Journal of Real-Time Image Processing and Technology (JORIT), Vol. 4, Issue 2, 2025

    Integrates blockchain and AI techniques to create a secure, low-latency IoT data exchange framework for industrial applications.

  • Machine Learning Techniques for Predicting Shelf Life of Perishable Food Items

    Published in: International Journal of Basic and Applied Sciences (IJBAS), 2025

    Applies regression and deep learning models to predict food shelf life, improving supply chain optimization and waste reduction.

  • Enhancing Network Intrusion Detection using Genetic Algorithm and Fuzzy Logic

    Published in: International Journal of Electronics and Reliability Engineering (IJERIA), Vol. 12, Issue 1, 2019

    Proposes a hybrid fuzzy-genetic model for intrusion detection systems, improving accuracy and response time in large-scale networks.

  • Optimization of Data Mining Algorithms using Swarm Intelligence

    Published in: International Journal of Mathematics and Soft Computing (IJMSEA), Vol. 12, Issue 3, 2018

    Explores swarm intelligence-based optimization methods for improving efficiency in data mining and pattern recognition algorithms.
